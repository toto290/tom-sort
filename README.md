# TomSort
Tom's image sorting app

Target Funcionality:
- SortingMode
    - presents all images in the working folder one by one
    - user can select individual tag for grouping (eg.: "ChristmasDinner2009")
    - defines and offers recommended tags (based on meta data or earlier choices)
    - renames image files in a structured way
    - moves/copies image files to tag-folders
    - user can directly delete unwanted images
    
- TwinFindingMode
    - looks for similar pictures for the user to choose which to keep
    - similarity is based on meta data and colour patterns
